===============================================================================
                        WHAT YOU NEED TO DO
===============================================================================

5 Manual Steps (~30 minutes total):

1. CREATE RENDER SERVICE (5-10 min)
   → Go to: https://dashboard.render.com
   → New Web Service from GitHub
   → Select: fdkng/SHOPBRAIN_AI
   → Root Directory: backend
   → Add environment variables from backend/.env
   → Deploy
   
2. APPLY SUPABASE MIGRATIONS (2 min)
   → Go to: https://supabase.com/dashboard
   → SQL Editor → New Query
   → Paste: backend/supabase_schema.sql
   → Run

3. CONFIGURE STRIPE WEBHOOK (3 min)
   → Go to: https://dashboard.stripe.com/developers/webhooks
   → Add endpoint: https://shopbrain-backend-xxxxx.onrender.com/webhook
   → Events: checkout.session.completed, customer.subscription.updated
   → Copy signing secret → Add to Render env var

4. DEPLOY FRONTEND (5 min)
   → Update: frontend/.env (VITE_API_BASE=render-url)
   → Build: cd frontend && npm run build
   → Deploy: npx vercel (or use GitHub integration)

5. TEST (5 min)
   → Visit frontend URL
   → Sign up
   → Test checkout
   → Verify Supabase records
